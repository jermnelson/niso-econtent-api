Path to Resource

Comments from Best Practice Review Topic ID #1

Path from finding a resource to accessing it should be intuitive, seamless and offer few/no barriers to the end user. The user should not have to leave the environment in which the resource was found. The resource should be displayed within the library catalogue/discovery layer and should not be taken to a third party website, e.g., publisher or aggregator site.

FASTEN Alignment with NCIP

  • A POST request with an NCIP Item Id or Request Id and returns optional data by requesting agent. Aligns with NCIP 5.3.2 Lookup Item Service
